# Arabic numbers to roman numerals

## Release / Update date
**30/01/2020**

## Description
Write a function `arabicToRoman` that takes a number and returns its roman numeral as a string.

Example:
```javascript
arabicToRoman(3);
// returns:
'III'

arabicToRoman(24);
// returns:
'XXIV'
```

- Pass all the tests
- To run the tests: `mocha myFile.test.js`
